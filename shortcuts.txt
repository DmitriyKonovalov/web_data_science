Посмотреть токен
http POST http://127.0.0.1:8000/api/v1/auth/api-token-auth/ username="admin" password="adminadmin"

Показать все анализы пользователя
http http://127.0.0.1:8000/api/v1/analyses/ Authorization:"Token 7a49f7823afc726ecfcad6e6d2d38c675e627128"

Создать анализ
http POST http://localhost:8000/api/v1/analyses/ name="Analysis1" ws="WS_1" wd="WD_1" wd_start=100 wd_stop=200 wd_step=10 ws_start=5 ws_stop=15 Authorization:"Token 7a49f7823afc726ecfcad6e6d2d38c675e627128"

Загрузить файл с анализом
http POST http://localhost:8000/api/v1/analyses/5/load_file_data/ file_path="C:\data.csv" Authorization:"Token 7a49f7823afc726ecfcad6e6d2d38c675e627128"

Выполнить анализ
http POST http://localhost:8000/api/v1/analyses/5/execute/ Authorization:"Token 7a49f7823afc726ecfcad6e6d2d38c675e627128"

Скачать анализ (недоступно в консоли, но ссылка рабочая)
http POST http://localhost:8000/api/v1/analyses/6/download/ Authorization:"Token 7a49f7823afc726ecfcad6e6d2d38c675e627128"
